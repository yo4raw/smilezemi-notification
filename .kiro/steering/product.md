# Product Overview

スマイルゼミ「みまもるネット」の自動クローリングおよびLINE通知システム。GitHub Actionsで定期的に実行され、サイト情報の変更を検出してLINEに通知します。

## Core Capabilities

- **自動ログイン**: GitHub Secretsに保存された認証情報を使用してみまもるネットに自動ログイン
- **情報クローリング**: ログイン後のサイト内情報を自動的に抽出して構造化データとして保存
- **変更検出**: 前回実行時のデータと比較して変更を自動検出
- **LINE通知**: クローリング結果や変更内容をLINE Notify APIを通じて即座に通知
- **スケジュール実行**: GitHub Actionsのcronスケジュールまたは手動トリガーで柔軟に実行

## Target Use Cases

- 保護者が子供の学習状況や通知を定期的に自動確認
- サイト更新の自動監視とリアルタイム通知
- 手動チェックの負担を軽減する自動化ワークフロー

## Value Proposition

- **セキュア**: 認証情報をGitHub Secretsで安全に管理、ログにマスキング
- **自動化**: 定期実行により手動操作を不要に
- **即時通知**: LINE統合により情報をリアルタイムで受信
- **信頼性**: エラーハンドリングとリトライ機能で安定した動作
- **透明性**: GitHub Actions UIで実行ログを完全に可視化
